什麼是開放原始碼/自由軟體

在西元 1984 史托曼 (Richard Stallman) 創辦自由軟體基金會 FSF (Free Software Foundation)，提倡自由軟體運動，直到這個時候，程式碼開放的這些軟體有了第一次的正式名稱 - 自由軟體 (Free Software)。 要特別理解的是 Free 指的是「自由」並非「免費」。自由軟體基金會所制定自由軟體的四個自由如下:

  1. 可以在任何目的之下自由地使用這些程式
  2. 可以自由地研讀這些程式是如何運作的，並且各取所需
  3. 可以自由地在散佈拷貝版，以協助週遭的人
  4. 可以自由的修改程式，並且將改善的部份公開發表，受惠整個社會

濱江國小自由軟體

濱江國小過去受惠於眾多自由軟體的成果，並得到自由軟體社群的諸多幫忙，資訊業務才得以順利推展。今天很高興的宣布濱江國小成為自由軟體社群的一份子，我們將整理自己內部開發的成果並以自由軟體授權釋出，以達到教學資源共享的目的。

我們的主要使用的版控系統為 git ，整理完的程式碼皆會陸續釋出至 http://github.com/bjes。

As always, patches welcome. :)
